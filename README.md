Анализ датасета, содержащего информацию о сне, физической активности, стрессе и других факторах образа жизни.

Цели:
1. Провести полный EDA анализ
2. Выявить закономерности между сном и образом жизни
3. Визуализировать результаты
4. Создать интерактивный дашборд в Yandex DataLens
5. Прикрепить результаты на GitHub

Вывод: люди с проблемами веса и расстройствами сна склонны спать меньше.
